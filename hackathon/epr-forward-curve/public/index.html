    How the score works. Forward Score = climate × (0.5·pending-pressure + 0.3·conversion-room + 0.2·state-propensity), where pressure is in-flight bills vs. the busiest state, room is pending relative to what's already enacted, propensity is how hard the state over-indexes on circular-economy lawmaking vs. its peer baseline (/insights/state-gap), and climate is a national multiplier from the advances-vs-weakens momentum (/bills/stance-momentum).
    Honesty notes. This is a transparent heuristic, not a trained model — it ranks relative likelihood, not a calendar date. Pipeline counts come from /bills/map-summary (all public/free tier, no key). Momentum is the noisiest classifier axis, so it only dampens the score (0.6–1.0×), never dominates it.
